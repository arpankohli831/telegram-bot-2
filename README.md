# 🤖 Telegram Earning & Stock Bot (Node.js)

A **Telegram bot built with Node.js & Telegraf** that supports:
- 💰 Manual UPI payments (QR code + UTR)
- 👥 Referral earnings
- 📦 Stock management (Admin only)
- 🎁 Promo codes (one-time use)
- ⭐ Paid Push system
- 📢 Channel & Owner contact buttons
- 🔐 Admin approval system
- 🌐 Free 24/7 hosting support

---

## ✨ Features

### 👤 User Features
- Add funds using **UPI QR code**
- Send **UTR or payment screenshot**
- Earn ₹0.5 per referral
- Use promo codes (one-time)
- View available stocks
- Paid Push options
- Contact owner directly
- Join official channel

### 👑 Admin Features
- Approve payments manually
- Add promo codes
- Add stocks
- Full balance control
- Secure admin-only commands

---

## 🛠 Tech Stack

- **Node.js 18+**
- **Telegraf (Telegram Bot API)**
- **SQLite3** (lightweight database)
- **Railway / VPS compatible**
